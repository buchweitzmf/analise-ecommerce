# 🛒 Análise de Vendas — E-commerce Amazon

## Sobre o Projeto
Análise exploratória de dados de vendas de uma loja Amazon indiana, 
com foco em identificar padrões de vendas por categoria, região e 
comportamento dos pedidos.

## Ferramentas Utilizadas
- Python + Pandas
- Matplotlib
- Jupyter Notebook
- Power BI

## Perguntas Respondidas
- Qual categoria de produto mais vende?
- Como as vendas evoluíram ao longo do tempo?
- Quais os estados que mais compram?
- Qual o status de pedido mais comum?

## Principais Insights
- **Set** é a categoria com maior volume de vendas, seguida de Kurta — ambas roupas tradicionais indianas
- O pico de vendas ocorreu em **abril de 2022**, com queda gradual nos meses seguintes
- **Maharashtra** é o estado que mais compra, com quase o dobro de Karnataka em segundo lugar
- **Shipped** é o status mais comum, mas a taxa de cancelamento de **14,2%** é relativamente alta — um ponto de atenção para o negócio

## Dataset
Dataset disponível no Kaggle: [E-Commerce Sales Dataset](https://www.kaggle.com/datasets/thedevastator/unlock-profits-with-e-commerce-sales-data)

## Como Executar
1. Clone o repositório
2. Abra o arquivo `ecommerce.ipynb` no Jupyter
3. Execute as células em ordem
4. O dashboard está disponível em `analise_ecommerce.pbix`
